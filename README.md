# 简介

> 项目名称:双港中学官网的后端管理

> 技术栈:nodeJS,express4,swig, multer, mongoose, body-parser

> 项目功能:
    + 静态页面的实现
    + 登录，注册的实现
    + 后端登录，注册逻辑处理
    + 后端导航的添加、修改、删除实现
    + 首页轮播图的上传和后端管理图片逻辑处理
    + 后端轮播图的添加、删除、修改实现
    + 详情页的实现
    + 后端分页功能实现

> 总结：
    + 在获取 file 路径的时候获取到的路径不是绝对路径
  解决方法：利用 multer 实现上传并把路径放到数据库，然后拿到
  路径并返回给前端渲染
    + 在 post 请求时不能直接拿到数据
  i. 解决方法:body-parser 的中间件，用 req.body 拿到数据
